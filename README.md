A modern, responsive food and recipe website showcasing trending recipes, restaurants, and foodie experiences. Built with HTML and CSS
with a focus on mobile-first design, interactive cards, hover effects, and smooth navigation.

Features

Responsive layout optimized for desktop and mobile
Recipe cards in grid format with hover glow effects
Filter recipes based on cuisine type
Smooth anchor jump navigation, accounting for fixed navbar
Interactive buttons, filters, and menus
Modern styling with CSS grid, flexbox, and transitions
Supports Dark Mode with a toggle button



Technologies Used

HTML5 – semantic markup for content
CSS3 – responsive styling, hover effects, grids, and flex layouts
VS Code – development environment
